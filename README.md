# 🌾 India Agriculture Crop Production Analysis

## 📌 Project Overview

This project presents a comprehensive **data-driven analysis of agricultural crop production in India** using **Exploratory Data Analysis (EDA)** and **Machine Learning techniques**.

The analysis is performed using Python in the Spyder IDE to extract insights and build predictive models.

---

## 📊 Dataset Information

* **Source:** Government of India ([data.gov.in]
* **Total Records:** 345,000+
* **Features:**

  * State
  * District
  * Crop
  * Season
  * Area
  * Production
  * Yield

---

## 🛠️ Technologies Used

* **Programming:** Python
* **Libraries:** Pandas, NumPy
* **Visualization:** Matplotlib, Seaborn
* **Machine Learning:** Scikit-learn (Linear Regression)
* **IDE:** Spyder

---

## 🎯 Project Objectives

* 📍 State-wise Total Production Analysis
* 🌦️ Season-wise Production Comparison
* 🌱 Top Crops by Production Volume
* 📈 Year-wise Production Trends
* 🔗 Correlation Between Area & Production
* 📦 Outlier Detection using Boxplots
* 🗺️ District-level Analysis
* 🤖 ML Model: Crop Production Prediction

## 🚀 How to Run

1. Clone the repository
2. Install dependencies:

   ```
   pip install -r requirements.txt
   ```
3. Open the Python file in Spyder:

   ```
   crop_analysis.py
   ```
4. Run the script
